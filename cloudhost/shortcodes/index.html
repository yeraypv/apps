Bootstrap3 Shortcodes

Bootstrap Shortcodes

This theme comes with Bootstrap 3 shortcode plugin for WordPress to use the latest base features and components of Bootstrap 3 framework. You can instantly add Bootstrap buttons, tables, tabs, icons, columns and other components into your content via WordPress editor.

Example Shortcode 1

[bs_title_line line="yes"]Example Shortcode[/bs_title_line]

Example Shortcode 2

[bs_column_wrap]
[bs_column size="6"]
Column 1 Content
[/bs_column]
[bs_column size="6"]
Column 2 Content
[/bs_column][/bs_column_wrap]
